# Contributing to entfirst

Thank you for your interest in contributing to entfirst!

## How to Contribute

1. **Fork** the repository and create your branch from `main`.
2. **Make your changes** with clear, descriptive commit messages.
3. **Test** your changes before submitting.
4. **Open a Pull Request** using the provided PR template.

## Code Style

- Keep commits focused and atomic.
- Write clear commit messages that describe the change.

## Reporting Issues

Use the GitHub Issues templates to report bugs or request features.

## Questions

For questions, open a GitHub Issue or reach out via [enterprisefirst.ai](https://enterprisefirst.ai).

# entfirst

**enterprisefirst.ai** — Enterprise First AB

## Overview

entfirst is the foundation repository for [Enterprise First AB](https://enterprisefirst.ai). This repository serves as the template and source of truth for the project.

## Getting Started

Clone this repository:

```bash
git clone https://github.com/maticyorg/entfirst.git
cd entfirst
```
